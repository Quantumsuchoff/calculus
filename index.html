§ 1

Polar ↔ Rectangular Coordinate Conversions

↑ top
Core Conversion Identities
\(x = r\cos\theta\)   \(y = r\sin\theta\) rectangular from polar
\(r^2 = x^2 + y^2\)   \(\tan\theta = \dfrac{y}{x}\) polar from rectangular
\(r = \sqrt{x^2+y^2}\)   \(\theta = \arctan\!\left(\dfrac{y}{x}\right) + \text{quadrant adjust}\)

Algorithm A — Rectangular \((x,y)\) → Polar \((r,\theta)\)

Step-by-Step Procedure
  1. Compute \(r = \sqrt{x^2 + y^2}\).
  2. Compute reference angle \(\alpha = \arctan\!\left(\dfrac{|y|}{|x|}\right)\).
  3. Assign \(\theta\) by quadrant of \((x,y)\):
    • Q I: \(\theta = \alpha\)
    • Q II: \(\theta = \pi - \alpha\)
    • Q III: \(\theta = \pi + \alpha\)
    • Q IV: \(\theta = 2\pi - \alpha\)
  4. Write answer as \((r,\,\theta)\).
  5. Check: verify \(x = r\cos\theta\) and \(y = r\sin\theta\) reproduce the originals.
⚠ Quadrant Rule: \(\arctan(y/x)\) always returns a value in \(\left(-\tfrac{\pi}{2},\tfrac{\pi}{2}\right)\). Always inspect the signs of \(x\) and \(y\) to place \(\theta\) in the correct quadrant — the calculator alone cannot do this for you.
Example — Convert \((-3,\,3)\) to Polar (Algorithm A)

Given: \((x,y) = (-3,\,3)\). We apply Algorithm A in order.

[Alg A, Step 1] Compute \(r\).
\[r = \sqrt{x^2 + y^2} = \sqrt{(-3)^2 + 3^2} = \sqrt{9 + 9} = \sqrt{18} = 3\sqrt{2}\]

We always take \(r > 0\) here.

[Alg A, Step 2] Compute the reference angle \(\alpha\) using absolute values.
\[\alpha = \arctan\!\left(\frac{|y|}{|x|}\right) = \arctan\!\left(\frac{3}{3}\right) = \arctan(1) = \frac{\pi}{4}\]

The reference angle is \(\alpha = \tfrac{\pi}{4}\) (i.e., \(45^\circ\)). This is just the acute angle to the nearest axis — it does not yet account for which quadrant we are in.

[Alg A, Step 3] Determine the quadrant and assign \(\theta\).

Inspect the signs: \(x = -3 < 0\) and \(y = 3 > 0\). This places the point in Quadrant II.

From the SOP table: for Q II, \(\theta = \pi - \alpha\).

\[\theta = \pi - \frac{\pi}{4} = \frac{4\pi}{4} - \frac{\pi}{4} = \frac{3\pi}{4}\]
[Alg A, Step 4] Write the final answer.
\[(r,\,\theta) = \left(3\sqrt{2},\;\frac{3\pi}{4}\right)\]
[Alg A, Step 5] Verify by converting back to rectangular.
\[x = r\cos\theta = 3\sqrt{2}\cdot\cos\!\left(\tfrac{3\pi}{4}\right) = 3\sqrt{2}\cdot\left(-\tfrac{\sqrt{2}}{2}\right) = -3 \quad\checkmark\]
\[y = r\sin\theta = 3\sqrt{2}\cdot\sin\!\left(\tfrac{3\pi}{4}\right) = 3\sqrt{2}\cdot\tfrac{\sqrt{2}}{2} = 3 \quad\checkmark\]

Both coordinates match the original point. The answer is confirmed.

Algorithm B — Polar \((r,\theta)\) → Rectangular \((x,y)\)

Step-by-Step Procedure
  1. Compute \(x = r\cos\theta\).
  2. Compute \(y = r\sin\theta\).
  3. Simplify. Done.
Example — Convert \(\left(4,\,\tfrac{2\pi}{3}\right)\) to Rectangular (Algorithm B)

Given: \((r,\theta) = \!\left(4,\,\tfrac{2\pi}{3}\right)\). Apply Algorithm B directly.

[Alg B, Step 1] Compute \(x\).
\[x = r\cos\theta = 4\cos\!\left(\frac{2\pi}{3}\right) = 4\cdot\left(-\frac{1}{2}\right) = -2\]
[Alg B, Step 2] Compute \(y\).
\[y = r\sin\theta = 4\sin\!\left(\frac{2\pi}{3}\right) = 4\cdot\frac{\sqrt{3}}{2} = 2\sqrt{3}\]
[Alg B, Step 3] Simplify and state the answer.
\[(x,y) = \left(-2,\;2\sqrt{3}\right)\]

Note: \(\cos\!\left(\tfrac{2\pi}{3}\right) = -\tfrac{1}{2}\) because \(\tfrac{2\pi}{3}\) is in Q II; the reference angle is \(\tfrac{\pi}{3}\) and cosine is negative in Q II.

§ 2

Polar ↔ Algebraic Equation Conversions

↑ top

Algorithm C — Rectangular Equation → Polar

Step-by-Step Procedure
  1. First, scan for \(x^2+y^2\) and replace directly with \(r^2\).
  2. Replace every remaining \(x\) with \(r\cos\theta\) and every \(y\) with \(r\sin\theta\).
  3. Simplify and factor out \(r\) where possible.
  4. Divide both sides by \(r\) (valid when \(r\neq 0\)) or multiply by \(r\) if needed.
  5. Solve for \(r\) (or leave as an implicit polar equation).
Tip: Always scan for \(x^2+y^2\) before touching individual \(x\) or \(y\) terms. One substitution \(x^2+y^2\to r^2\) can cut the algebra in half. If you expand a binomial, look again for \(x^2+y^2\) in the result before proceeding.
Example — Problem 23a: \(x^2 + (y-4)^2 = 16\) (Algorithm C)

Observation: There is no \(x^2+y^2\) visible yet. The squared binomial must be expanded first, then we look for it.

[Alg C, Step 1] Expand the binomial and scan for \(x^2+y^2\).
\[x^2 + (y-4)^2 = 16\]
\[x^2 + y^2 - 8y + 16 = 16 \qquad \text{expand }(y-4)^2 = y^2 - 8y + 16\]
\[x^2 + y^2 - 8y = 0 \qquad \text{subtract 16 from both sides}\]

Now \(x^2+y^2\) appears. Replace it with \(r^2\):

\[r^2 - 8y = 0\]
[Alg C, Step 2] Replace the remaining \(y\) with \(r\sin\theta\).
\[r^2 - 8r\sin\theta = 0\]
[Alg C, Step 3] Factor out \(r\).
\[r\bigl(r - 8\sin\theta\bigr) = 0\]

This gives \(r = 0\) (the pole, included in any curve passing through the origin) or \(r - 8\sin\theta = 0\).

[Alg C, Step 4] Divide by \(r\) (drop the trivial solution \(r=0\)).
\[r = 8\sin\theta\]
[Alg C, Step 5] Already solved for \(r\).
\[r = 8\sin\theta\]

Sanity check: This is standard form \(r = a\sin\theta\), which represents a circle centered at \((0,\,4)\) with radius \(4\) — consistent with the original equation.

Example — Problem 23b: \((x-7)^2 + y^2 = 49\) (Algorithm C)
[Alg C, Step 1] Expand and look for \(x^2 + y^2\).
\[(x-7)^2 + y^2 = 49\]
\[x^2 - 14x + 49 + y^2 = 49\]
\[x^2 + y^2 - 14x = 0\]

Replace \(x^2+y^2 \to r^2\):

\[r^2 - 14x = 0\]
[Alg C, Step 2] Replace the remaining \(x\) with \(r\cos\theta\).
\[r^2 - 14r\cos\theta = 0\]
[Alg C, Step 3] Factor.
\[r(r - 14\cos\theta) = 0\]
[Alg C, Step 4] Divide by \(r\).
\[r = 14\cos\theta\]

Sanity check: Standard form \(r=a\cos\theta\) → circle centered at \((7,0)\) with radius 7. Matches original.

Example — Problem 23c: \(x^2 + y^2 = 16x\) (Algorithm C)
[Alg C, Step 1] \(x^2+y^2\) is already visible on the left. Immediately replace it.
\[r^2 = 16x\]
[Alg C, Step 2] Replace \(x\) with \(r\cos\theta\).
\[r^2 = 16r\cos\theta\]
[Alg C, Step 3–4] The right side already has a factor of \(r\). Divide both sides by \(r\).
\[r = 16\cos\theta\]
Example — Problem 23d: \(x^2 + y^2 - 24y = 0\) (Algorithm C)
[Alg C, Step 1] \(x^2+y^2\) is already visible. Replace it.
\[r^2 - 24y = 0\]
[Alg C, Step 2] Replace \(y\) with \(r\sin\theta\).
\[r^2 - 24r\sin\theta = 0\]
[Alg C, Steps 3–5] Factor, divide by \(r\).
\[r = 24\sin\theta\]

Algorithm D — Polar Equation → Rectangular

Step-by-Step Procedure
  1. Multiply both sides by \(r\) if needed to create \(r^2\), \(r\cos\theta\), or \(r\sin\theta\) terms.
  2. Replace: \(r^2 \to x^2+y^2\), \(r\cos\theta \to x\), \(r\sin\theta \to y\).
  3. Simplify. If it is a circle, complete the square to find center and radius.
Common Polar Forms → Rectangular
\(r = a\cos\theta\) → circle: center \(\left(\tfrac{a}{2},0\right)\), radius \(\tfrac{|a|}{2}\)
\(r = a\sin\theta\) → circle: center \(\left(0,\tfrac{a}{2}\right)\), radius \(\tfrac{|a|}{2}\)
\(r = a\) → circle: centered at origin, radius \(a\)
\(\theta = c\) → line through origin at angle \(c\)
Example — Convert \(r = 6\sin\theta\) to Rectangular (Algorithm D)

Given: \(r = 6\sin\theta\). Notice \(r\sin\theta\) is not yet isolated because there is a bare \(r\) on the left.

[Alg D, Step 1] Multiply both sides by \(r\) to manufacture \(r^2\) and \(r\sin\theta\).
\[r \cdot r = r \cdot 6\sin\theta \;\Longrightarrow\; r^2 = 6r\sin\theta\]
[Alg D, Step 2] Replace \(r^2 \to x^2+y^2\) and \(r\sin\theta \to y\).
\[x^2 + y^2 = 6y\]
[Alg D, Step 3] Complete the square to identify the circle.
\[x^2 + y^2 - 6y = 0\]
\[x^2 + (y^2 - 6y + 9) = 9 \qquad\left(\text{add }\left(\tfrac{6}{2}\right)^2 = 9\text{ to both sides}\right)\]
\[x^2 + (y - 3)^2 = 9\]

Circle centered at \((0,\,3)\) with radius \(3\) — consistent with \(r=a\sin\theta\): center \(\!\left(0,\tfrac{a}{2}\right)=(0,3)\), radius \(\tfrac{a}{2}=3\). ✓

Algorithm E — Graphing a Polar Curve

Step-by-Step Procedure
  1. Classify using the table above.
  2. Find zeros: Set \(r=0\) and solve for \(\theta\).
  3. Find max \(|r|\): amplitude = leading coefficient.
  4. Build a table at key angles \(0,\tfrac{\pi}{6},\tfrac{\pi}{4},\tfrac{\pi}{3},\tfrac{\pi}{2},\ldots\)
  5. Check symmetry (replace \(\theta\to-\theta\); \(\theta\to\pi-\theta\); \(r\to-r\)).
  6. Plot and connect smoothly. Inner loops appear where \(r<0\).
Example — Problem 24: Graph \(r = 2\cos(3\theta)\) (Algorithm E)
[Alg E, Step 1] Classify the curve.

The equation has the form \(r = a\cos(n\theta)\) with \(a=2\) and \(n=3\) (odd). By the table: 3-petal rose curve.

[Alg E, Step 2] Find zeros (where the curve passes through the origin).
\[2\cos(3\theta) = 0 \;\Rightarrow\; \cos(3\theta) = 0 \;\Rightarrow\; 3\theta = \frac{\pi}{2}+k\pi \;\Rightarrow\; \theta = \frac{\pi}{6}+\frac{k\pi}{3},\quad k\in\mathbb{Z}\]

In \([0,\pi]\): \(\theta = \tfrac{\pi}{6},\,\tfrac{\pi}{2},\,\tfrac{5\pi}{6}\). These are the angles at which the curve crosses the origin (petal boundaries).

[Alg E, Step 3] Find maximum \(|r|\).

\(|r|_{\max} = |a| = 2\). The tips of the petals reach \(r = 2\).

[Alg E, Step 4] Build a table of key values.
\(\theta\)\(3\theta\)\(r = 2\cos(3\theta)\)
\(0\)\(0\)\(2\)
\(\pi/6\)\(\pi/2\)\(0\)
\(\pi/3\)\(\pi\)\(-2\)
\(\pi/2\)\(3\pi/2\)\(0\)
\(2\pi/3\)\(2\pi\)\(2\)
\(5\pi/6\)\(5\pi/2\)\(0\)
\(\pi\)\(3\pi\)\(-2\)

Note: Negative \(r\) values mean the point is plotted opposite the direction \(\theta\) points — this traces the petals on the other side of the pole.

[Alg E, Step 5] Check symmetry.

Replace \(\theta \to -\theta\): \(r = 2\cos(3(-\theta)) = 2\cos(-3\theta) = 2\cos(3\theta)\) — unchanged. Symmetric about the polar axis.

[Alg E, Step 6] Sketch.

Three petals: one centered on the positive \(x\)-axis at \(\theta=0\), one at \(\theta=\tfrac{2\pi}{3}\), one at \(\theta=\tfrac{4\pi}{3}\). Each petal tip is at \(r=2\).

Result: 3-petal rose, max radius 2, petal tips at \(\theta = 0,\,\tfrac{2\pi}{3},\,\tfrac{4\pi}{3}\).

Example — Problem 25: Graph \(r = 2\sin(2\theta)\) (Algorithm E)
[Alg E, Step 1] Classify.

Form is \(r = a\sin(n\theta)\) with \(a=2\), \(n=2\) (even). By the table: 4-petal rose (\(2n = 4\) petals).

[Alg E, Step 2] Find zeros.
\[2\sin(2\theta)=0 \;\Rightarrow\; \sin(2\theta)=0 \;\Rightarrow\; 2\theta = k\pi \;\Rightarrow\; \theta = 0,\,\frac{\pi}{2},\,\pi,\,\frac{3\pi}{2}\]
[Alg E, Step 3] Max \(|r| = 2\) (petal tips).
[Alg E, Step 4] Table of key values.
\(\theta\)\(2\theta\)\(r = 2\sin(2\theta)\)
\(0\)\(0\)\(0\)
\(\pi/4\)\(\pi/2\)\(2\)
\(\pi/2\)\(\pi\)\(0\)
\(3\pi/4\)\(3\pi/2\)\(-2\)
\(\pi\)\(2\pi\)\(0\)
\(5\pi/4\)\(5\pi/2\)\(2\)
\(3\pi/2\)\(3\pi\)\(0\)
\(7\pi/4\)\(7\pi/2\)\(-2\)
[Alg E, Step 5] Symmetry.

Replace \(\theta\to\pi-\theta\): \(\sin(2(\pi-\theta))=\sin(2\pi-2\theta)=\sin(-2\theta)=-\sin(2\theta)\). So \(r\to-r\), which traces the same curve. Symmetric about \(\theta=\tfrac{\pi}{2}\) and the polar axis.

[Alg E, Step 6] Four petals with tips at \(\theta = \tfrac{\pi}{4},\,\tfrac{3\pi}{4},\,\tfrac{5\pi}{4},\,\tfrac{7\pi}{4}\), each reaching \(|r|=2\).
Rose Curve Rules
\(r = a\cos(n\theta)\): \(\;n\) odd → \(n\) petals; \(n\) even → \(2n\) petals
Petal tips at \(n\theta = 0,\,\pi,\,2\pi,\ldots\) (cosine); \(n\theta = \tfrac{\pi}{2},\,\tfrac{3\pi}{2},\ldots\) (sine)
§ 4

Areas of Polar Regions

↑ top
Master Area Formulas
\(\displaystyle A = \frac{1}{2}\int_{\alpha}^{\beta} r^2\,d\theta\) area of one polar region
\(\displaystyle A = \frac{1}{2}\int_{\alpha}^{\beta}\!\left[r_{\text{outer}}^2 - r_{\text{inner}}^2\right]d\theta\) area between two curves
Essential Trig Identities for Integrating \(r^2\)
\(\cos^2\theta = \dfrac{1+\cos(2\theta)}{2}\)
\(\sin^2\theta = \dfrac{1-\cos(2\theta)}{2}\)
\(\cos^2(n\theta) = \dfrac{1+\cos(2n\theta)}{2}\)

Algorithm F — Area Enclosed by a Single Polar Curve

Step-by-Step Procedure
  1. Identify \(r = f(\theta)\).
  2. Find limits \(\alpha\) and \(\beta\) by setting \(r=0\) and solving.
  3. Use symmetry if applicable (integrate half, then double).
  4. Set up \(\displaystyle A = \frac{1}{2}\int_{\alpha}^{\beta} r^2\,d\theta\).
  5. Expand \(r^2\) and apply power-reduction identities for \(\cos^2\) or \(\sin^2\) terms.
  6. Integrate term by term and evaluate.

Algorithm G — Area of ONE Petal (Rose Curve)

Step-by-Step Procedure
  1. Set \(r=0\): \(a\cos(n\theta)=0 \Rightarrow \theta = \tfrac{\pi}{2n}\). One petal spans \(\left[-\tfrac{\pi}{2n},\,\tfrac{\pi}{2n}\right]\).
  2. By symmetry: \(\displaystyle A_{\text{petal}} = 2\cdot\frac{1}{2}\int_{0}^{\pi/(2n)} r^2\,d\theta = \int_{0}^{\pi/(2n)} a^2\cos^2(n\theta)\,d\theta\).
  3. Apply \(\cos^2(n\theta) = \tfrac{1+\cos(2n\theta)}{2}\), integrate, evaluate.
  4. Total area = (number of petals) \(\times\; A_{\text{petal}}\).
Example — Total Area Enclosed by \(r = 2\cos(3\theta)\) [Problem 24] (Algorithms F & G)

Goal: Find the total area enclosed by the 3-petal rose \(r = 2\cos(3\theta)\).

Strategy: Use Algorithm G — compute the area of one petal, then multiply by 3.

[Alg G, Step 1] Find the angular bounds of one petal. Set \(r = 0\):
\[2\cos(3\theta) = 0 \;\Rightarrow\; \cos(3\theta) = 0 \;\Rightarrow\; 3\theta = \frac{\pi}{2} \;\Rightarrow\; \theta = \frac{\pi}{6}\]

One petal sweeps from \(\theta = -\tfrac{\pi}{6}\) to \(\theta = \tfrac{\pi}{6}\) (symmetric about \(\theta=0\)).

[Alg G, Step 2] Apply symmetry — integrate from \(0\) to \(\tfrac{\pi}{6}\) and double.
\[A_{\text{petal}} = 2\cdot\frac{1}{2}\int_{0}^{\pi/6} r^2\,d\theta = \int_{0}^{\pi/6} \bigl[2\cos(3\theta)\bigr]^2\,d\theta = \int_{0}^{\pi/6} 4\cos^2(3\theta)\,d\theta\]
[Alg G, Step 3] Apply the power-reduction identity \(\cos^2(3\theta) = \dfrac{1+\cos(6\theta)}{2}\).
\[A_{\text{petal}} = \int_{0}^{\pi/6} 4\cdot\frac{1+\cos(6\theta)}{2}\,d\theta = \int_{0}^{\pi/6} 2\bigl[1 + \cos(6\theta)\bigr]\,d\theta\]

Split the integral:

\[A_{\text{petal}} = 2\int_{0}^{\pi/6} 1\,d\theta + 2\int_{0}^{\pi/6}\cos(6\theta)\,d\theta\]

Evaluate each part:

\[2\int_{0}^{\pi/6} 1\,d\theta = 2\left[\theta\right]_0^{\pi/6} = 2\cdot\frac{\pi}{6} = \frac{\pi}{3}\]
\[2\int_{0}^{\pi/6}\cos(6\theta)\,d\theta = 2\left[\frac{\sin(6\theta)}{6}\right]_0^{\pi/6} = \frac{1}{3}\left[\sin(\pi) - \sin(0)\right] = 0\]

So: \(A_{\text{petal}} = \dfrac{\pi}{3} + 0 = \dfrac{\pi}{3}\)

[Alg G, Step 4] Multiply by the number of petals (3).
\[A_{\text{total}} = 3 \cdot \frac{\pi}{3} = \pi\]

Algorithm H — Inner Loop Area (Limaçon)

Step-by-Step Procedure
  1. Confirm \(\tfrac{a}{b} < 1\) (inner loop exists).
  2. Find bounds: set \(r=0\), solve \(a + b\cos\theta=0 \Rightarrow \cos\theta = -\tfrac{a}{b}\). Call solutions \(\theta_1,\theta_2\).
  3. The inner loop is traced on \([\theta_1, \theta_2]\) where \(r \le 0\).
  4. Set up: \(\displaystyle A_{\text{inner}} = \frac{1}{2}\int_{\theta_1}^{\theta_2} r^2\,d\theta\).
  5. Expand and integrate using trig identities.
Example — Inner Loop Area of \(r = 1 + 2\cos\theta\) [Problem 26a] (Algorithm H)

Given: \(r = 1 + 2\cos\theta\). Here \(a = 1\), \(b = 2\).

[Alg H, Step 1] Check for inner loop: \(\dfrac{a}{b} = \dfrac{1}{2} < 1\). Yes, an inner loop exists.
[Alg H, Step 2] Find the angular bounds of the inner loop. Set \(r = 0\):
\[1 + 2\cos\theta = 0 \;\Rightarrow\; \cos\theta = -\frac{1}{2} \;\Rightarrow\; \theta = \frac{2\pi}{3} \text{ and } \theta = \frac{4\pi}{3}\]

The inner loop is traced for \(\theta \in \left[\tfrac{2\pi}{3},\,\tfrac{4\pi}{3}\right]\) (where \(r\le 0\)).

[Alg H, Step 3] Set up the integral. Note that \(r^2 \ge 0\) regardless of the sign of \(r\).
\[A_{\text{inner}} = \frac{1}{2}\int_{2\pi/3}^{4\pi/3} (1+2\cos\theta)^2\,d\theta\]
[Alg H, Step 4] Expand \((1+2\cos\theta)^2 = 1 + 4\cos\theta + 4\cos^2\theta\).

Apply \(\cos^2\theta = \tfrac{1+\cos(2\theta)}{2}\):

\[= 1 + 4\cos\theta + 4\cdot\frac{1+\cos(2\theta)}{2} = 1 + 4\cos\theta + 2 + 2\cos(2\theta) = 3 + 4\cos\theta + 2\cos(2\theta)\]
[Alg H, Step 5] Integrate term by term.
\[A_{\text{inner}} = \frac{1}{2}\int_{2\pi/3}^{4\pi/3}\!\bigl[3 + 4\cos\theta + 2\cos(2\theta)\bigr]\,d\theta\]
\[= \frac{1}{2}\left[3\theta + 4\sin\theta + \sin(2\theta)\right]_{2\pi/3}^{4\pi/3}\]
\[= \frac{1}{2}\!\left[\left(4\pi + 4\sin\tfrac{4\pi}{3}+\sin\tfrac{8\pi}{3}\right) - \left(2\pi + 4\sin\tfrac{2\pi}{3}+\sin\tfrac{4\pi}{3}\right)\right]\]
\[= \frac{1}{2}\!\left[\left(4\pi - 2\sqrt{3} - \tfrac{\sqrt{3}}{2}\right) - \left(2\pi + 2\sqrt{3} + \left(-\tfrac{\sqrt{3}}{2}\right)\right)\right]\]
\[= \frac{1}{2}\!\left[2\pi - 4\sqrt{3}\right]\]
\[A_{\text{inner}} = \pi - 2\sqrt{3}\]

Algorithm I — Outer Loop & Area Between Loops

Step-by-Step Procedure
  1. \(A_{\text{total}} = \dfrac{1}{2}\displaystyle\int_0^{2\pi} r^2\,d\theta\)
  2. \(A_{\text{outer}} = A_{\text{total}} - A_{\text{inner}}\)
  3. \(A_{\text{between loops}} = A_{\text{outer}} - A_{\text{inner}}\)

Algorithm J — Area Between Two Polar Curves

Step-by-Step Procedure
  1. Find intersections: set \(r_1 = r_2\) and solve for \(\theta\). Also check if both pass through the origin.
  2. Determine which is the outer curve (larger \(|r|\)) on each sub-interval.
  3. Set up: \(\displaystyle A = \frac{1}{2}\int_{\alpha}^{\beta}\!\left[r_{\text{outer}}^2 - r_{\text{inner}}^2\right]d\theta\).
  4. Integrate and evaluate.
Example — Problem 27: Area Between \(r=3\) and \(r=3-3\cos\theta\) (Algorithm J)

Given: \(r_1 = 3\) (circle, radius 3 centered at origin) and \(r_2 = 3-3\cos\theta\) (cardioid, since \(\tfrac{a}{b}=1\)).

[Alg J, Step 1] Find intersections. Set \(r_1 = r_2\):
\[3 = 3 - 3\cos\theta \;\Rightarrow\; 0 = -3\cos\theta \;\Rightarrow\; \cos\theta = 0 \;\Rightarrow\; \theta = \frac{\pi}{2},\;\frac{3\pi}{2}\]
[Alg J, Step 2] Determine which curve is outer on each interval.

On \(\theta \in \left[-\tfrac{\pi}{2},\,\tfrac{\pi}{2}\right]\) (right half, where \(\cos\theta > 0\)): \(3-3\cos\theta < 3\), so \(r_1=3\) is outer.

On \(\theta \in \left[\tfrac{\pi}{2},\,\tfrac{3\pi}{2}\right]\) (left half, where \(\cos\theta < 0\)): \(3-3\cos\theta > 3\), so \(r_2=3-3\cos\theta\) is outer.

[Alg J, Steps 3–4]

(b) Inside \(r=3\), outside the cardioid — \(\theta\in[-\tfrac{\pi}{2},\tfrac{\pi}{2}]\), outer \(= r=3\):

\[A = \frac{1}{2}\int_{-\pi/2}^{\pi/2}\!\left[9 - (3-3\cos\theta)^2\right]d\theta\]

Expand: \((3-3\cos\theta)^2 = 9 - 18\cos\theta + 9\cos^2\theta\), so:

\[9 - (9 - 18\cos\theta + 9\cos^2\theta) = 18\cos\theta - 9\cos^2\theta\]

Apply \(\cos^2\theta = \tfrac{1+\cos(2\theta)}{2}\):

\[= 18\cos\theta - \frac{9}{2}(1+\cos(2\theta))\]

Integrate using symmetry about \(\theta=0\) (double the integral from \(0\) to \(\tfrac{\pi}{2}\)):

\[A = \frac{1}{2}\cdot 2\int_{0}^{\pi/2}\!\left[18\cos\theta - \frac{9}{2} - \frac{9}{2}\cos(2\theta)\right]d\theta\]
\[= \left[18\sin\theta - \frac{9}{2}\theta - \frac{9}{4}\sin(2\theta)\right]_0^{\pi/2}\]
\[= \left(18 - \frac{9\pi}{4} - 0\right) - (0)\]
\[A_b = 18 - \frac{9\pi}{4}\]

(c) Inside cardioid, outside \(r=3\) — \(\theta\in[\tfrac{\pi}{2},\tfrac{3\pi}{2}]\), outer \(= r=3-3\cos\theta\):

\[A = \frac{1}{2}\int_{\pi/2}^{3\pi/2}\!\left[(3-3\cos\theta)^2 - 9\right]d\theta\]

By the same expansion (sign reversed):

\[(3-3\cos\theta)^2 - 9 = -18\cos\theta + 9\cos^2\theta = -18\cos\theta + \frac{9}{2}(1+\cos(2\theta))\]
\[A = \frac{1}{2}\left[-18\sin\theta + \frac{9}{2}\theta + \frac{9}{4}\sin(2\theta)\right]_{\pi/2}^{3\pi/2}\]
\[= \frac{1}{2}\!\left[\left(18 + \frac{27\pi}{4} + 0\right)-\left(-18+\frac{9\pi}{4}+0\right)\right] = \frac{1}{2}\left[36 + \frac{18\pi}{4}\right]\]
\[A_c = 18 + \frac{9\pi}{4}\]

Parametric Curves ↔ Algebraic Equations

↑ top

Algorithm K — Eliminate the Parameter

Choose the Right Strategy
  1. Strategy 1 — Direct substitution: Solve one equation for \(t\), substitute.
  2. Strategy 2 — Pythagorean identity: If \(x=a\cos t\), \(y=b\sin t\): \(\left(\tfrac{x}{a}\right)^2+\left(\tfrac{y}{b}\right)^2=1\).
  3. Strategy 3 — Secant/Tangent identity: If \(x=\sec^2 t\), use \(\sec^2 t - \tan^2 t = 1\).
  4. Strategy 4 — Exponential: If \(x=e^t\), \(y=e^{kt}\): \(y=x^k\).
  5. Strategy 5 — Power: If \(x=t^m\), \(y=t^n\): \(y=x^{n/m}\).
  6. Always state the restriction on \(x\) from the parameter domain.
Example — Problem 29a: \(x(t)=\sec^2 t,\;\; y(t)=6\tan t\) (Strategy 3)

Strategy selection: \(x\) contains \(\sec^2 t\) and \(y\) contains \(\tan t\). This signals Strategy 3 (secant/tangent identity: \(\sec^2 t - \tan^2 t = 1\)).

[Alg K, Step 1] Identify the trig functions present.
\[\sec^2 t = x \qquad \tan t = \frac{y}{6} \;\Rightarrow\; \tan^2 t = \frac{y^2}{36}\]
[Alg K, Step 3] Apply the identity \(\sec^2 t - \tan^2 t = 1\). Substitute:
\[x - \frac{y^2}{36} = 1\]

Solve for \(x\) to write in standard form:

\[x = 1 + \frac{y^2}{36}\]
[Alg K, Step 6] State the restriction. Since \(\sec^2 t \ge 1\) for all \(t\), we have \(x \ge 1\).
\[x = 1 + \frac{y^2}{36}, \quad x \ge 1\]

Shape: Parabola opening to the right with vertex at \((1,0)\).

Example — Problem 29b: \(x(t)=-1+4e^t,\;\; y(t)=2+e^{2t}\) (Strategy 4)

Strategy selection: Both \(x\) and \(y\) involve \(e^t\) and \(e^{2t}\) — notice \(e^{2t}=(e^t)^2\). Use Strategy 4 (exponential).

[Alg K, Step 1] Isolate \(e^t\) from the \(x\) equation.
\[x = -1 + 4e^t \;\Rightarrow\; 4e^t = x+1 \;\Rightarrow\; e^t = \frac{x+1}{4}\]
[Alg K, Step 4] Express \(e^{2t}\) in terms of \(x\):
\[e^{2t} = \left(e^t\right)^2 = \left(\frac{x+1}{4}\right)^2 = \frac{(x+1)^2}{16}\]

Substitute into \(y\):

\[y = 2 + e^{2t} = 2 + \frac{(x+1)^2}{16}\]
[Alg K, Step 6] State the restriction. Since \(e^t > 0\) for all \(t\), we need \(\dfrac{x+1}{4} > 0\), so \(x > -1\).
\[y = 2 + \frac{(x+1)^2}{16}, \quad x > -1\]

Shape: Parabola opening upward with vertex at \((-1, 2)\).

Example — Problem 30a: \(x(t)=-1+3\cos t,\;\; y(t)=1+2\sin t\) (Strategy 2)

Strategy selection: \(x\) contains \(\cos t\) and \(y\) contains \(\sin t\) — shifted sinusoidal form. Use Strategy 2 (Pythagorean identity \(\cos^2 t + \sin^2 t = 1\)).

[Alg K, Step 1] Isolate \(\cos t\) and \(\sin t\).
\[x = -1 + 3\cos t \;\Rightarrow\; \cos t = \frac{x+1}{3}\]
\[y = 1 + 2\sin t \;\Rightarrow\; \sin t = \frac{y-1}{2}\]
[Alg K, Step 2] Apply \(\cos^2 t + \sin^2 t = 1\). Substitute:
\[\left(\frac{x+1}{3}\right)^2 + \left(\frac{y-1}{2}\right)^2 = 1\]
\[\frac{(x+1)^2}{9} + \frac{(y-1)^2}{4} = 1\]
[Alg K, Step 6] No restriction — \(t \in [0,2\pi)\) traces the full ellipse.

Shape: Ellipse centered at \((-1,\,1)\), semi-major axis \(a=3\) (horizontal), semi-minor axis \(b=2\) (vertical).

Extra Example: \(x(t) = t^2,\;\; y(t) = t^6 + 3\) (Strategy 5)
[Alg K, Step 5] Express \(t\) in terms of \(x\): \(x = t^2 \Rightarrow t^2 = x\).

Notice \(t^6 = (t^2)^3 = x^3\). Substitute into \(y\):

\[y = t^6 + 3 = x^3 + 3\]
[Alg K, Step 6] Restriction: \(x = t^2 \ge 0\).
\[y = x^3 + 3, \quad x \ge 0\]
Extra Example: \(x(t) = e^t,\;\; y(t) = e^{5t} - 4\) (Strategy 4)
[Alg K, Step 4] From \(x = e^t\), note \(e^{5t} = (e^t)^5 = x^5\). Substitute:
\[y = x^5 - 4\]
[Alg K, Step 6] Since \(e^t > 0\), we need \(x > 0\).
\[y = x^5 - 4, \quad x > 0\]

Parametrization of Geometric Objects

↑ top

Algorithm L — Parametrize a Line Segment from \(P_1\) to \(P_2\)

Step-by-Step Procedure
  1. Let \(P_1 = (x_1,y_1)\) be the start and \(P_2=(x_2,y_2)\) be the end.
  2. Write with \(t\in[0,1]\): \(\;x(t) = x_1 + (x_2-x_1)t\) and \(y(t) = y_1+(y_2-y_1)t\).
  3. Verify: at \(t=0\) you get \(P_1\); at \(t=1\) you get \(P_2\). ✓
Example — Problem 28a: Line Segment from \((1,6)\) to \((-3,1)\) (Algorithm L)

Given: \(P_1 = (1,6)\), \(P_2 = (-3,1)\).

[Alg L, Step 1] Identify the start and end points.
\[x_1 = 1,\quad y_1 = 6,\quad x_2 = -3,\quad y_2 = 1\]

Compute the displacement in each direction:

\[x_2 - x_1 = -3 - 1 = -4 \qquad y_2 - y_1 = 1 - 6 = -5\]
[Alg L, Step 2] Write the parametric equations.
\[x(t) = 1 + (-4)t = 1 - 4t\]
\[y(t) = 6 + (-5)t = 6 - 5t\]
[Alg L, Step 3] Verify endpoints.
\[t=0:\quad x(0)=1,\; y(0)=6 \;\Rightarrow\; P_1=(1,6) \quad\checkmark\]
\[t=1:\quad x(1)=1-4=-3,\; y(1)=6-5=1 \;\Rightarrow\; P_2=(-3,1) \quad\checkmark\]
\[x(t) = 1-4t,\quad y(t) = 6-5t,\quad t\in[0,1]\]
Extra Example: Line Segment from \((-2,3)\) to \((5,-1)\) (Algorithm L)
[Alg L, Step 1] \(P_1=(-2,3)\), \(P_2=(5,-1)\).
\[x_2-x_1 = 5-(-2)=7 \qquad y_2-y_1 = -1-3=-4\]
[Alg L, Step 2]
\[x(t) = -2 + 7t,\qquad y(t) = 3 - 4t\]
[Alg L, Step 3] Verify: \(t=0 \to (-2,3)\) ✓; \(t=1 \to (5,-1)\) ✓.
\[x(t) = -2+7t,\quad y(t) = 3-4t,\quad t\in[0,1]\]

Algorithm M — Parametrize a Circle or Ellipse

Standard Forms
  1. Circle \(x^2+y^2=r^2\): \(\;x(t)=r\cos t,\;\;y(t)=r\sin t,\;\;t\in[0,2\pi]\)
  2. Shifted circle \((x-h)^2+(y-k)^2=r^2\): \(\;x(t)=h+r\cos t,\;\;y(t)=k+r\sin t\)
  3. Ellipse — first rewrite as \(\dfrac{x^2}{a^2}+\dfrac{y^2}{b^2}=1\): then \(x(t)=a\cos t,\;\;y(t)=b\sin t,\;\;t\in[0,2\pi]\)
  4. Shifted ellipse: \(\;x(t)=h+a\cos t,\;\;y(t)=k+b\sin t\)
Key step: Always divide the ellipse equation by the constant on the right to get \(=1\) before reading off \(a\) and \(b\). Reading \(a^2\) and \(b^2\) from an un-normalized equation gives wrong values.
Example — Problem 28b: \(x^2 + y^2 = 4\) (Algorithm M)
[Alg M, Step 1] Identify the form: \(x^2+y^2=4 = 2^2\). This matches Form 1 (circle centered at origin). Read off \(r = 2\).
[Alg M, Step 2] Write the parametric equations using Form 1.
\[x(t) = 2\cos t,\quad y(t) = 2\sin t,\quad t\in[0,2\pi]\]

Verification: \(x(t)^2 + y(t)^2 = 4\cos^2 t + 4\sin^2 t = 4(\cos^2 t + \sin^2 t) = 4\) ✓

Example — Problem 28c: \(25x^2 + 9y^2 = 225\) (Algorithm M)
[Alg M, Step 1] Not in standard form. Divide both sides by 225.
\[\frac{25x^2}{225} + \frac{9y^2}{225} = 1 \;\Rightarrow\; \frac{x^2}{9} + \frac{y^2}{25} = 1\]

This matches Form 3 (ellipse). Read off: \(a^2 = 9 \Rightarrow a = 3\) and \(b^2 = 25 \Rightarrow b = 5\).

[Alg M, Step 2] Write the parametric equations using Form 3.
\[x(t) = 3\cos t,\quad y(t) = 5\sin t,\quad t\in[0,2\pi]\]

Verification: \(\dfrac{(3\cos t)^2}{9} + \dfrac{(5\sin t)^2}{25} = \cos^2 t + \sin^2 t = 1\) ✓

Extra Example: \(9x^2 + 16y^2 = 144\) (Algorithm M)
[Alg M, Step 1] Divide by 144:
\[\frac{x^2}{16} + \frac{y^2}{9} = 1 \;\Rightarrow\; a^2=16\Rightarrow a=4,\quad b^2=9\Rightarrow b=3\]

Note: \(a\) goes with \(x\), \(b\) goes with \(y\) — match them to their respective variables.

[Alg M, Step 2]
\[x(t) = 4\cos t,\quad y(t) = 3\sin t,\quad t\in[0,2\pi]\]

Verification: \(\tfrac{16\cos^2 t}{16} + \tfrac{9\sin^2 t}{9} = \cos^2 t + \sin^2 t = 1\) ✓

§ 7

Tangent Lines & Slopes of Parametric Curves

↑ top
Parametric Derivative Formula
\(\displaystyle\frac{dy}{dx} = \frac{dy/dt}{dx/dt}\) requires \(dx/dt \neq 0\)
Horizontal tangent: \(\dfrac{dy}{dt}=0\) AND \(\dfrac{dx}{dt}\neq 0\)
Vertical tangent: \(\dfrac{dx}{dt}=0\) AND \(\dfrac{dy}{dt}\neq 0\)

Algorithm N — Tangent Line at a Specific \(t\)-Value

Step-by-Step Procedure
  1. Differentiate: compute \(\dfrac{dx}{dt}\) and \(\dfrac{dy}{dt}\).
  2. Evaluate both at \(t_0\); compute slope \(m = \dfrac{dy/dt}{dx/dt}\big|_{t=t_0}\).
  3. Find the point: \((x_0,y_0) = (x(t_0),\,y(t_0))\).
  4. Write the tangent line: \(y - y_0 = m(x-x_0)\).
Example — Problem 31a: \(x(t)=4t^3-12t,\;\;y(t)=t^2+3\) at \(t=2\) (Algorithm N)
[Alg N, Step 1] Differentiate both parametric equations with respect to \(t\).
\[\frac{dx}{dt} = \frac{d}{dt}\!\left[4t^3 - 12t\right] = 12t^2 - 12\]
\[\frac{dy}{dt} = \frac{d}{dt}\!\left[t^2 + 3\right] = 2t\]
[Alg N, Step 2] Evaluate both derivatives at \(t_0 = 2\) and compute the slope.
\[\frac{dx}{dt}\bigg|_{t=2} = 12(2)^2 - 12 = 12(4) - 12 = 48 - 12 = 36\]
\[\frac{dy}{dt}\bigg|_{t=2} = 2(2) = 4\]

Apply the parametric derivative formula:

\[m = \frac{dy/dt}{dx/dt}\bigg|_{t=2} = \frac{4}{36} = \frac{1}{9}\]

Note: \(dx/dt = 36 \neq 0\), so the tangent is not vertical and the formula is valid.

[Alg N, Step 3] Find the point \((x_0, y_0)\) by evaluating the position at \(t=2\).
\[x(2) = 4(2)^3 - 12(2) = 4(8) - 24 = 32 - 24 = 8\]
\[y(2) = (2)^2 + 3 = 4 + 3 = 7\]

Point: \((x_0,y_0) = (8,7)\).

[Alg N, Step 4] Write the equation of the tangent line using point-slope form.
\[y - y_0 = m(x - x_0) \;\Rightarrow\; y - 7 = \frac{1}{9}(x - 8)\]
\[y - 7 = \frac{1}{9}(x - 8)\]
Example — Problem 30b: \(x(t)=-1+3\cos t,\;\;y(t)=1+2\sin t\) at \(t=\tfrac{\pi}{4}\) (Algorithm N)
[Alg N, Step 1] Differentiate.
\[\frac{dx}{dt} = -3\sin t \qquad \frac{dy}{dt} = 2\cos t\]
[Alg N, Step 2] Evaluate at \(t_0 = \tfrac{\pi}{4}\). Recall \(\sin\tfrac{\pi}{4} = \cos\tfrac{\pi}{4} = \tfrac{\sqrt{2}}{2}\).
\[\frac{dx}{dt}\bigg|_{t=\pi/4} = -3\cdot\frac{\sqrt{2}}{2} = -\frac{3\sqrt{2}}{2}\]
\[\frac{dy}{dt}\bigg|_{t=\pi/4} = 2\cdot\frac{\sqrt{2}}{2} = \sqrt{2}\]

Compute slope:

\[m = \frac{\sqrt{2}}{-\,3\sqrt{2}/2} = \sqrt{2}\cdot\frac{-2}{3\sqrt{2}} = -\frac{2}{3}\]
[Alg N, Step 3] Find the point.
\[x\!\left(\tfrac{\pi}{4}\right) = -1 + 3\cos\tfrac{\pi}{4} = -1 + 3\cdot\frac{\sqrt{2}}{2} = -1 + \frac{3\sqrt{2}}{2}\]
\[y\!\left(\tfrac{\pi}{4}\right) = 1 + 2\sin\tfrac{\pi}{4} = 1 + 2\cdot\frac{\sqrt{2}}{2} = 1 + \sqrt{2}\]
[Alg N, Step 4] Write the tangent line.
\[y - (1+\sqrt{2}) = -\frac{2}{3}\!\left(x - \left(-1+\frac{3\sqrt{2}}{2}\right)\right)\]

Algorithm O — Find ALL Horizontal & Vertical Tangent Points

Step-by-Step Procedure
  1. Compute \(\dfrac{dx}{dt}\) and \(\dfrac{dy}{dt}\).
  2. Horizontal: Solve \(\dfrac{dy}{dt}=0\). For each solution \(t_0\), confirm \(\dfrac{dx}{dt}\big|_{t_0}\neq 0\). Report \((x(t_0),y(t_0))\).
  3. Vertical: Solve \(\dfrac{dx}{dt}=0\). For each solution \(t_0\), confirm \(\dfrac{dy}{dt}\big|_{t_0}\neq 0\). Report \((x(t_0),y(t_0))\).
  4. Always report the actual \((x,y)\) coordinate pair, not just \(t\).
⚠ Caution: If \(\dfrac{dx}{dt}=0\) AND \(\dfrac{dy}{dt}=0\) at the same \(t\), the tangent is indeterminate — this is a cusp. Do not report it as horizontal or vertical.
Example — Problem 31b & c: \(x(t)=4t^3-12t,\;\;y(t)=t^2+3\) — All Tangent Points (Algorithm O)

Goal: Find all horizontal and vertical tangent points.

[Alg O, Step 1] Differentiate.
\[\frac{dx}{dt} = 12t^2 - 12 = 12(t^2-1) = 12(t-1)(t+1)\]
\[\frac{dy}{dt} = 2t\]
[Alg O, Step 2] Find horizontal tangents — set \(\dfrac{dy}{dt}=0\).
\[2t = 0 \;\Rightarrow\; t = 0\]

Confirm: \(\dfrac{dx}{dt}\big|_{t=0} = 12(0-1)(0+1) = -12 \neq 0\) ✓ (valid horizontal tangent, not a cusp)

Find the point:

\[x(0) = 4(0)^3 - 12(0) = 0 \qquad y(0) = (0)^2 + 3 = 3\]

Horizontal tangent at: \((0,\,3)\)

[Alg O, Step 3] Find vertical tangents — set \(\dfrac{dx}{dt}=0\).
\[12(t-1)(t+1) = 0 \;\Rightarrow\; t = 1 \quad\text{or}\quad t = -1\]

At \(t = 1\):

Confirm: \(\dfrac{dy}{dt}\big|_{t=1} = 2(1) = 2 \neq 0\) ✓

\[x(1) = 4(1)^3 - 12(1) = 4-12 = -8 \qquad y(1) = (1)^2+3 = 4\]

Vertical tangent at: \((-8,\,4)\)

At \(t = -1\):

Confirm: \(\dfrac{dy}{dt}\big|_{t=-1} = 2(-1) = -2 \neq 0\) ✓

\[x(-1) = 4(-1)^3 - 12(-1) = -4+12 = 8 \qquad y(-1) = (-1)^2+3 = 4\]

Vertical tangent at: \((8,\,4)\)

[Alg O, Step 4] Summary of all tangent points.
Type\(t\) valuePoint \((x,y)\)
Horizontal\(t=0\)\((0,\,3)\)
Vertical\(t=1\)\((-8,\,4)\)
Vertical\(t=-1\)\((8,\,4)\)
Example — Problem 30c: \(x(t)=-1+3\cos t,\;\;y(t)=1+2\sin t,\;\;t\in[0,2\pi)\) (Algorithm O)
[Alg O, Step 1] Differentiate.
\[\frac{dx}{dt} = -3\sin t \qquad \frac{dy}{dt} = 2\cos t\]
[Alg O, Step 2] Horizontal tangents — set \(\dfrac{dy}{dt}=0\).
\[2\cos t = 0 \;\Rightarrow\; \cos t = 0 \;\Rightarrow\; t = \frac{\pi}{2},\;\frac{3\pi}{2}\]

At \(t = \tfrac{\pi}{2}\): Confirm \(\dfrac{dx}{dt} = -3\sin\tfrac{\pi}{2} = -3 \neq 0\) ✓

\[x\!\left(\tfrac{\pi}{2}\right) = -1+3\cos\tfrac{\pi}{2} = -1+0 = -1 \qquad y\!\left(\tfrac{\pi}{2}\right) = 1+2\sin\tfrac{\pi}{2} = 1+2 = 3\]

Horizontal tangent at \((-1,\,3)\).

At \(t = \tfrac{3\pi}{2}\): Confirm \(\dfrac{dx}{dt} = -3\sin\tfrac{3\pi}{2} = 3 \neq 0\) ✓

\[x\!\left(\tfrac{3\pi}{2}\right) = -1+3\cos\tfrac{3\pi}{2} = -1+0 = -1 \qquad y\!\left(\tfrac{3\pi}{2}\right) = 1+2\sin\tfrac{3\pi}{2} = 1-2 = -1\]

Horizontal tangent at \((-1,\,-1)\).

Horizontal tangent points: \((-1,\,3)\) and \((-1,\,-1)\)

[Alg O, Step 3] Vertical tangents — set \(\dfrac{dx}{dt}=0\).
\[-3\sin t = 0 \;\Rightarrow\; \sin t = 0 \;\Rightarrow\; t = 0,\;\pi\]

At \(t = 0\): Confirm \(\dfrac{dy}{dt} = 2\cos 0 = 2 \neq 0\) ✓

\[x(0) = -1+3\cos 0 = -1+3 = 2 \qquad y(0) = 1+2\sin 0 = 1\]

Vertical tangent at \((2,\,1)\).

At \(t = \pi\): Confirm \(\dfrac{dy}{dt} = 2\cos\pi = -2 \neq 0\) ✓

\[x(\pi) = -1+3\cos\pi = -1-3 = -4 \qquad y(\pi) = 1+2\sin\pi = 1\]

Vertical tangent at \((-4,\,1)\).

Vertical tangent points: \((2,\,1)\) and \((-4,\,1)\)

[Alg O, Step 4] Summary.

Horizontal\(t=3\pi/2\)\((-1,\,-1)\)
Vertical\(t=0\)\((2,\,1)\)
Vertical\(t=\pi\)\((-4,\,1)\)

Extra Example: \(x(t)=2t^3-6t,\;\;y(t)=t^2-4t\) at \(t=3\) (Algorithm N)
[Alg N, Step 1] Differentiate.
\[\frac{dx}{dt} = 6t^2 - 6 \qquad \frac{dy}{dt} = 2t - 4\]

\[m = \frac{2}{48} = \frac{1}{24}\]

\[x(3) = 2(27)-6(3) = 54-18 = 36 \qquad y(3) = 9-12 = -3\]
