§ 1

Polar ↔ Rectangular Coordinate Conversions

Core Conversion Identities
\(x = r\cos\theta\) \(y = r\sin\theta\) rectangular from polar
\(r^2 = x^2 + y^2\) \(\tan\theta = \dfrac{y}{x}\) polar from rectangular
\(r = \sqrt{x^2+y^2}\) \(\theta = \arctan\!\left(\dfrac{y}{x}\right)\) + quadrant adjust

Algorithm A — Rectangular \((x,y)\) → Polar \((r,\theta)\)

Step-by-Step Procedure
  1. Compute \(r = \sqrt{x^2 + y^2}\).
  2. Compute reference angle \(\alpha = \arctan\!\left(\dfrac{|y|}{|x|}\right)\).
  3. Assign \(\theta\) by quadrant of \((x,y)\):
    • Q I: \(\theta = \alpha\)
    • Q II: \(\theta = \pi - \alpha\)
    • Q III: \(\theta = \pi + \alpha\)
    • Q IV: \(\theta = 2\pi - \alpha\)
  4. Write answer as \((r,\,\theta)\).
  5. Check: verify \(x = r\cos\theta\) and \(y = r\sin\theta\) reproduce the originals.
⚠ Quadrant Rule: \(\arctan(y/x)\) always returns a value in \(\left(-\tfrac{\pi}{2},\tfrac{\pi}{2}\right)\). Always inspect the signs of \(x\) and \(y\) to place \(\theta\) in the correct quadrant.

Algorithm B — Polar \((r,\theta)\) → Rectangular \((x,y)\)

Step-by-Step Procedure
  1. Compute \(x = r\cos\theta\).
  2. Compute \(y = r\sin\theta\).
  3. Simplify. Done.
Example — Convert \((-3,\,3)\) to polar

\(r = \sqrt{(-3)^2 + 3^2} = \sqrt{18} = 3\sqrt{2}\)

\(\alpha = \arctan\!\left(\tfrac{3}{3}\right) = \arctan(1) = \tfrac{\pi}{4}\)

Point is in Q II \((x<0,\, y>0)\) \(\Rightarrow\) \(\theta = \pi - \tfrac{\pi}{4} = \tfrac{3\pi}{4}\)

Answer: \(\left(3\sqrt{2},\;\tfrac{3\pi}{4}\right)\)

§ 2

Polar ↔ Algebraic Equation Conversions

Algorithm C — Rectangular Equation → Polar

Step-by-Step Procedure
  1. First, replace any \(x^2+y^2\) directly with \(r^2\) (biggest shortcut).
  2. Replace every remaining \(x\) with \(r\cos\theta\) and every \(y\) with \(r\sin\theta\).
  3. Simplify and factor out \(r\) where possible.
  4. Divide both sides by \(r\) (valid when \(r\neq 0\)) or multiply by \(r\) if \(r\) is in the denominator.
  5. Solve for \(r\) (or leave as an implicit polar equation).
Tip: Always scan the equation for \(x^2+y^2\) before touching individual \(x\) or \(y\) terms. A single substitution \(x^2+y^2\to r^2\) can cut the algebra in half.
Example — Problem 23a: \(x^2 + (y-4)^2 = 16\)

Expand: \(x^2 + y^2 - 8y + 16 = 16\)

Simplify: \(x^2 + y^2 - 8y = 0\)

Replace \(x^2+y^2 \to r^2\): \(\;r^2 - 8y = 0\)

Replace \(y \to r\sin\theta\): \(\;r^2 - 8r\sin\theta = 0\)

Factor: \(r(r - 8\sin\theta) = 0\)

Answer: \(r = 8\sin\theta\)

Example — Problem 23b: \((x-7)^2 + y^2 = 49\)

Expand: \(x^2 - 14x + 49 + y^2 = 49 \;\Rightarrow\; x^2+y^2 - 14x = 0\)

Replace: \(r^2 - 14r\cos\theta = 0\)

Factor & divide by \(r\): \(r = 14\cos\theta\)

Example — Problem 23c: \(x^2 + y^2 = 16x\)

Replace \(x^2+y^2\to r^2\) and \(x\to r\cos\theta\): \(\;r^2 = 16r\cos\theta\)

Divide by \(r\): \(r = 16\cos\theta\)

Example — Problem 23d: \(x^2 + y^2 - 24y = 0\)

Replace: \(r^2 - 24r\sin\theta = 0 \;\Rightarrow\; r(r-24\sin\theta)=0\)

Answer: \(r = 24\sin\theta\)

Algorithm D — Polar Equation → Rectangular

Step-by-Step Procedure
  1. Multiply both sides by \(r\) if needed to create \(r^2\), \(r\cos\theta\), or \(r\sin\theta\) terms.
  2. Replace: \(r^2 \to x^2+y^2\), \(\;r\cos\theta \to x\), \(\;r\sin\theta \to y\).
  3. Simplify. If it's a circle, complete the square to find center and radius.
Common Polar Forms → Rectangular
\(r = a\cos\theta\) → circle: center \(\left(\tfrac{a}{2},0\right)\), radius \(\tfrac{|a|}{2}\)
\(r = a\sin\theta\) → circle: center \(\left(0,\tfrac{a}{2}\right)\), radius \(\tfrac{|a|}{2}\)
\(r = a\) → circle: centered at origin, radius \(a\)
\(\theta = c\) → line through origin at angle \(c\)

Identifying & Graphing Polar Curves

Equation FormCurve TypeKey FeaturesWorksheet Examples
\(r = a\) Circle Centered at origin, radius \(a\) Prob 27: \(r=3\)
\(r = a\cos\theta\) or \(a\sin\theta\) Circle Passes through origin 22A, 22F
\(r = a \pm b\cos\theta,\;\tfrac{a}{b}=1\) Cardioid Heart-shaped, passes through origin 22B, 22C
\(r = a \pm b\cos\theta,\;\tfrac{a}{b}>1\) Limaçon (no inner loop) Dimpled or convex —
\(r = a \pm b\cos\theta,\;\tfrac{a}{b}<1\) Limaçon with inner loop Inner loop where \(r < 0\) 26a, 26b
\(r = a\cos(n\theta)\) or \(a\sin(n\theta)\) Rose / Flower \(n\) petals if \(n\) odd; \(2n\) petals if \(n\) even 22D, Probs 24 & 25

Algorithm E — Graphing a Polar Curve

Step-by-Step Procedure
  1. Classify the curve using the table above.
  2. Find zeros: Set \(r=0\) and solve for \(\theta\). These are the angles where the curve passes through the origin.
  3. Find max \(|r|\): The amplitude is the coefficient \(a\) in front of the trig function.
  4. Build a table of \((\theta, r)\) at \(\theta = 0,\,\tfrac{\pi}{6},\,\tfrac{\pi}{4},\,\tfrac{\pi}{3},\,\tfrac{\pi}{2},\,\ldots,\,2\pi\).
  5. Check symmetry:
    • Replace \(\theta \to -\theta\): if \(r\) unchanged → symmetric about polar axis.
    • Replace \(\theta \to \pi-\theta\): if \(r\) unchanged → symmetric about \(\theta=\tfrac{\pi}{2}\).
    • Replace \(r \to -r\): if equation unchanged → symmetric about the pole.
  6. Plot the \((r,\theta)\) pairs and connect smoothly.
  7. Inner loops appear when \(r < 0\) — the curve traces behind the pole.

Rose Curve Details — Problems 24 & 25

Rose Curve Rules
\(r = a\cos(n\theta)\): \(\;n\) odd → \(n\) petals; \(n\) even → \(2n\) petals
Petal tips at \(n\theta = 0,\,\pi,\,2\pi,\ldots\) (for cosine form), where \(|r| = a\)
Petal tip angles for sine: \(n\theta = \tfrac{\pi}{2},\,\tfrac{3\pi}{2},\ldots\)

Problem 24: \(r = 2\cos(3\theta)\) → 3-petal rose (\(n=3\), odd). Petals centered at \(\theta = 0,\,\tfrac{2\pi}{3},\,\tfrac{4\pi}{3}\).

Problem 25: \(r = 2\sin(2\theta)\) → 4-petal rose (\(n=2\), even). Petals at \(\theta = \tfrac{\pi}{4},\,\tfrac{3\pi}{4},\,\tfrac{5\pi}{4},\,\tfrac{7\pi}{4}\).

§ 4

Areas of Polar Regions

Master Area Formulas
\(\displaystyle A = \frac{1}{2}\int_{\alpha}^{\beta} r^2\,d\theta\) area of one polar region
\(\displaystyle A = \frac{1}{2}\int_{\alpha}^{\beta}\!\left[r_{\text{outer}}^2 - r_{\text{inner}}^2\right]d\theta\) area between two curves
Essential Trig Identities for Integrating \(r^2\)
\(\cos^2\theta = \dfrac{1+\cos(2\theta)}{2}\)
\(\sin^2\theta = \dfrac{1-\cos(2\theta)}{2}\)
\(\cos^2(n\theta) = \dfrac{1+\cos(2n\theta)}{2}\)

Algorithm F — Area Enclosed by a Single Polar Curve

Step-by-Step Procedure
  1. Identify \(r = f(\theta)\).
  2. Find limits \(\alpha\) and \(\beta\) by setting \(r=0\) and solving — these are the boundary angles.
  3. Use symmetry if applicable (integrate half the interval, then double).
  4. Set up \(\displaystyle A = \frac{1}{2}\int_{\alpha}^{\beta} r^2\,d\theta\).
  5. Expand \(r^2\), apply power-reduction identities for any \(\cos^2\) or \(\sin^2\) terms.
  6. Integrate term by term and evaluate.

Algorithm G — Area of ONE Petal (Rose Curve)

Step-by-Step Procedure
  1. Set \(r=0\): \(a\cos(n\theta)=0 \Rightarrow n\theta = \tfrac{\pi}{2} \Rightarrow \theta = \tfrac{\pi}{2n}\). One petal spans \(\left[-\tfrac{\pi}{2n},\,\tfrac{\pi}{2n}\right]\).
  2. By symmetry: \(\displaystyle A_{\text{petal}} = 2\cdot\frac{1}{2}\int_{0}^{\pi/(2n)} r^2\,d\theta = \int_{0}^{\pi/(2n)} a^2\cos^2(n\theta)\,d\theta\).
  3. Apply \(\cos^2(n\theta) = \tfrac{1+\cos(2n\theta)}{2}\), integrate, evaluate.
  4. Total area = (number of petals) \(\times\; A_{\text{petal}}\).
Example — Total area enclosed by \(r = 2\cos(3\theta)\) [Problem 24]

One petal: \(\cos(3\theta)=0 \Rightarrow \theta = \tfrac{\pi}{6}\). Petal: \(\left[-\tfrac{\pi}{6},\,\tfrac{\pi}{6}\right]\).

\[A_{\text{petal}} = \int_0^{\pi/6} 4\cos^2(3\theta)\,d\theta = \int_0^{\pi/6} 2\bigl[1+\cos(6\theta)\bigr]\,d\theta\]
\[= \left[2\theta + \frac{\sin(6\theta)}{3}\right]_0^{\pi/6} = \frac{\pi}{3}\]

3 petals total → Total Area \(= 3\cdot\dfrac{\pi}{3} = \pi\)

Algorithm H — Inner Loop Area (Limaçon)

Step-by-Step Procedure
  1. Confirm \(\tfrac{a}{b} < 1\) (inner loop exists).
  2. Find bounds: set \(r=0\), e.g. \(a + b\cos\theta=0 \Rightarrow \cos\theta = -\tfrac{a}{b}\). Let solutions be \(\theta_1\) and \(\theta_2\) (the loop's angular span).
  3. The inner loop is traced on \([\theta_1, \theta_2]\) where \(r \le 0\).
  4. Set up: \(\displaystyle A_{\text{inner}} = \frac{1}{2}\int_{\theta_1}^{\theta_2} r^2\,d\theta\). Note \(r^2\) is always \(\ge 0\).
  5. Expand and integrate using trig identities.

Algorithm I — Outer Loop & Area Between Loops

Step-by-Step Procedure
  1. \(A_{\text{total}} = \dfrac{1}{2}\displaystyle\int_0^{2\pi} r^2\,d\theta\)
  2. \(A_{\text{outer}} = A_{\text{total}} - A_{\text{inner}}\)
  3. \(A_{\text{between loops}} = A_{\text{outer}} - A_{\text{inner}}\)

Algorithm J — Area Between Two Polar Curves [Problem 27]

Step-by-Step Procedure
  1. Find intersections: set \(r_1 = r_2\) and solve for \(\theta\). Also check whether both curves pass through the origin.
  2. Determine which is the outer curve (larger \(|r|\)) on each sub-interval.
  3. Set up: \(\displaystyle A = \frac{1}{2}\int_{\alpha}^{\beta}\!\left[r_{\text{outer}}^2 - r_{\text{inner}}^2\right]d\theta\).
  4. Integrate and evaluate.
Example — Problem 27: \(r = 3\) and \(r = 3 - 3\cos\theta\)

Intersection: \(3 = 3-3\cos\theta \Rightarrow \cos\theta = 0 \Rightarrow \theta = \tfrac{\pi}{2},\,\tfrac{3\pi}{2}\).

(a) Area between both curves:

\[A = \frac{1}{2}\int_{-\pi/2}^{\pi/2}\!\left[9 - (3-3\cos\theta)^2\right]d\theta\]

(b) Inside \(r=3\), outside \(r=3-3\cos\theta\): outer = \(r=3\) on \(\left[-\tfrac{\pi}{2},\tfrac{\pi}{2}\right]\)

\[A = \frac{1}{2}\int_{-\pi/2}^{\pi/2}\!\left[9 - (3-3\cos\theta)^2\right]d\theta\]

(c) Inside \(r=3-3\cos\theta\), outside \(r=3\): outer = \(r=3-3\cos\theta\) on \(\left[\tfrac{\pi}{2},\tfrac{3\pi}{2}\right]\)

\[A = \frac{1}{2}\int_{\pi/2}^{3\pi/2}\!\left[(3-3\cos\theta)^2 - 9\right]d\theta\]
§ 5

Parametric Curves ↔ Algebraic Equations

Algorithm K — Eliminate the Parameter

Choose the Right Strategy
  1. Strategy 1 — Direct substitution: Solve one equation for \(t\), substitute into the other.
  2. Strategy 2 — Pythagorean trig identity: If \(x = a\cos t\), \(y = b\sin t\), then \(\left(\tfrac{x}{a}\right)^2 + \left(\tfrac{y}{b}\right)^2 = \cos^2 t + \sin^2 t = 1\).
  3. Strategy 3 — Secant/Tangent identity: If \(x = \sec^2 t\) and \(y\) involves \(\tan t\), use \(\sec^2 t - \tan^2 t = 1\).
  4. Strategy 4 — Exponential: If \(x = e^t\) and \(y = e^{kt}\), then \(y = x^k\).
  5. Strategy 5 — Power: If \(x = t^m\) and \(y = t^n\), then \(y = x^{n/m}\).
  6. State restrictions on \(x\) and/or \(y\) imposed by the parameter domain.
Example — Problem 29a: \(x(t)=\sec^2 t,\;\; y(t)=6\tan t\)

From \(y\): \(\tan t = \tfrac{y}{6}\), so \(\tan^2 t = \tfrac{y^2}{36}\).

Identity: \(\sec^2 t - \tan^2 t = 1 \;\Rightarrow\; x - \tfrac{y^2}{36} = 1\)

Answer: \(x = 1 + \dfrac{y^2}{36}\)  (parabola, \(x \ge 1\))

Example — Problem 29b: \(x(t)=-1+4e^t,\;\; y(t)=2+e^{2t}\)

From \(x\): \(e^t = \tfrac{x+1}{4}\), so \(e^{2t} = \left(\tfrac{x+1}{4}\right)^2 = \tfrac{(x+1)^2}{16}\).

Substitute into \(y\): \(y = 2 + \dfrac{(x+1)^2}{16}\)  (parabola, \(x > -1\))

Example — Problem 30a: \(x(t)=-1+3\cos t,\;\; y(t)=1+2\sin t\)

Isolate trig: \(\cos t = \dfrac{x+1}{3}\), \(\quad\sin t = \dfrac{y-1}{2}\)

Apply \(\cos^2 t + \sin^2 t = 1\):

\[\frac{(x+1)^2}{9} + \frac{(y-1)^2}{4} = 1\]

Ellipse with center \((-1,\,1)\), \(a=3\), \(b=2\)

Extra Example: \(x(t) = t^2,\;\; y(t) = t^6 + 3\)

Note \(t^6 = (t^2)^3 = x^3\). So: \(y = x^3 + 3\)

Extra Example: \(x(t) = e^t,\;\; y(t) = e^{5t} - 4\)

\(e^{5t} = (e^t)^5 = x^5\). So: \(y = x^5 - 4\)

§ 6

Parametrization of Geometric Objects

Algorithm L — Parametrize a Line Segment from \(P_1\) to \(P_2\)

Step-by-Step Procedure
  1. Let \(P_1 = (x_1,\,y_1)\) (start) and \(P_2 = (x_2,\,y_2)\) (end).
  2. Write with \(t \in [0,1]\):
    \[x(t) = x_1 + (x_2 - x_1)\,t\]
    \[y(t) = y_1 + (y_2 - y_1)\,t\]
  3. Verify: at \(t=0\) you get \(P_1\); at \(t=1\) you get \(P_2\). ✓
Example — Problem 28a: Line segment from \((1,6)\) to \((-3,1)\)

\(x(t) = 1 + (-3-1)\,t = 1 - 4t\)

\(y(t) = 6 + (1-6)\,t = 6 - 5t\)

Answer: \(x(t) = 1-4t,\;\; y(t)=6-5t,\;\; t\in[0,1]\)

Extra Example: Line segment from \((-2,3)\) to \((5,-1)\)

\(x(t) = -2 + 7t,\;\; y(t) = 3 - 4t,\;\; t\in[0,1]\)

Algorithm M — Parametrize a Circle or Ellipse

Standard Forms
  1. Circle \(x^2+y^2=r^2\): \(\;x(t)=r\cos t,\;\;y(t)=r\sin t,\;\;t\in[0,2\pi]\)
  2. Circle \((x-h)^2+(y-k)^2=r^2\): \(\;x(t)=h+r\cos t,\;\;y(t)=k+r\sin t\)
  3. Ellipse: First rewrite as \(\dfrac{x^2}{a^2}+\dfrac{y^2}{b^2}=1\), then \(x(t)=a\cos t,\;\;y(t)=b\sin t,\;\;t\in[0,2\pi]\)
  4. Shifted ellipse \(\dfrac{(x-h)^2}{a^2}+\dfrac{(y-k)^2}{b^2}=1\): \(\;x(t)=h+a\cos t,\;\;y(t)=k+b\sin t\)
Key step: Always divide the ellipse equation by the constant on the right side first to get it into standard form before reading off \(a\) and \(b\).
Example — Problem 28b: \(x^2 + y^2 = 4\)

Circle, \(r=2\). \(x(t)=2\cos t,\;\;y(t)=2\sin t,\;\;t\in[0,2\pi]\)

Example — Problem 28c: \(25x^2 + 9y^2 = 225\)

Divide by 225: \(\;\dfrac{x^2}{9} + \dfrac{y^2}{25} = 1\) \(\Rightarrow a=3,\; b=5\)

\(x(t)=3\cos t,\;\;y(t)=5\sin t,\;\;t\in[0,2\pi]\)

Extra Example: \(9x^2 + 16y^2 = 144\)

Divide by 144: \(\;\dfrac{x^2}{16}+\dfrac{y^2}{9}=1\) \(\Rightarrow a=4,\; b=3\)

\(x(t)=4\cos t,\;\;y(t)=3\sin t,\;\;t\in[0,2\pi]\)

§ 7

Tangent Lines & Slopes of Parametric Curves

Parametric Derivative Formula
\(\displaystyle\frac{dy}{dx} = \frac{dy/dt}{dx/dt}\) slope of curve at parameter \(t\) (requires \(dx/dt \neq 0\))
Horizontal tangent: \(\dfrac{dy}{dt}=0\) AND \(\dfrac{dx}{dt}\neq 0\)
Vertical tangent: \(\dfrac{dx}{dt}=0\) AND \(\dfrac{dy}{dt}\neq 0\)

Algorithm N — Tangent Line at a Specific \(t\)-Value

Step-by-Step Procedure
  1. Differentiate: compute \(\dfrac{dx}{dt}\) and \(\dfrac{dy}{dt}\).
  2. Evaluate both at the given \(t_0\), then compute slope \(m = \dfrac{dy/dt}{dx/dt}\bigg|_{t=t_0}\).
  3. Find the point: \((x_0, y_0) = \bigl(x(t_0),\,y(t_0)\bigr)\).
  4. Write the tangent line: \(y - y_0 = m\,(x - x_0)\).
Example — Problem 31a: \(x(t)=4t^3-12t,\;\;y(t)=t^2+3\) at \(t=2\)

\(\dfrac{dx}{dt} = 12t^2-12\), \(\quad\dfrac{dy}{dt} = 2t\)

At \(t=2\): \(\dfrac{dx}{dt} = 12(4)-12 = 36\), \(\quad\dfrac{dy}{dt} = 4\)

\(m = \dfrac{4}{36} = \dfrac{1}{9}\)

Point: \(x(2)=32-24=8\), \(\;y(2)=4+3=7\) → \((8,7)\)

Tangent line: \(y - 7 = \dfrac{1}{9}(x-8)\)

Example — Problem 30b: \(x(t)=-1+3\cos t,\;\;y(t)=1+2\sin t\) at \(t=\tfrac{\pi}{4}\)

\(\dfrac{dx}{dt} = -3\sin t\), \(\quad\dfrac{dy}{dt} = 2\cos t\)

At \(t=\tfrac{\pi}{4}\): \(\dfrac{dx}{dt} = -\dfrac{3\sqrt{2}}{2}\), \(\quad\dfrac{dy}{dt} = \sqrt{2}\)

\(m = \dfrac{\sqrt{2}}{-3\sqrt{2}/2} = -\dfrac{2}{3}\)

Point: \(\left(-1+\dfrac{3\sqrt{2}}{2},\;1+\sqrt{2}\right)\)

Tangent line: \(y - (1+\sqrt{2}) = -\dfrac{2}{3}\!\left(x - \left(-1+\tfrac{3\sqrt{2}}{2}\right)\right)\)

Algorithm O — Find ALL Horizontal & Vertical Tangent Points

Step-by-Step Procedure
  1. Compute \(\dfrac{dx}{dt}\) and \(\dfrac{dy}{dt}\).
  2. Horizontal tangents: solve \(\dfrac{dy}{dt}=0\) for all \(t\) in the domain. For each \(t_0\), verify \(\dfrac{dx}{dt}\big|_{t_0} \neq 0\). Report \(\bigl(x(t_0), y(t_0)\bigr)\).
  3. Vertical tangents: solve \(\dfrac{dx}{dt}=0\) for all \(t\) in the domain. For each \(t_0\), verify \(\dfrac{dy}{dt}\big|_{t_0} \neq 0\). Report \(\bigl(x(t_0), y(t_0)\bigr)\).
  4. Always report the actual coordinate pair \((x, y)\), not just \(t\).
⚠ Caution: If both \(\dfrac{dx}{dt}=0\) AND \(\dfrac{dy}{dt}=0\) at the same \(t\), the tangent is indeterminate. This is a cusp — require L'Hôpital or higher-order analysis.
Example — Problem 31b & c: \(x(t)=4t^3-12t,\;\;y(t)=t^2+3\)

\(\dfrac{dx}{dt} = 12t^2 - 12 = 12(t-1)(t+1)\), \(\quad\dfrac{dy}{dt} = 2t\)


Horizontal tangents (\(dy/dt=0\)): \(2t=0 \Rightarrow t=0\).

Check: \(dx/dt\big|_{t=0} = -12 \neq 0\) ✓

Point: \(x(0)=0,\;y(0)=3\) → \((0,\,3)\)


Vertical tangents (\(dx/dt=0\)): \(t=1\) or \(t=-1\).

At \(t=1\): \(dy/dt=2\neq 0\) ✓ → \(x(1)=4-12=-8,\;y(1)=4\) → \((-8,\,4)\)

At \(t=-1\): \(dy/dt=-2\neq 0\) ✓ → \(x(-1)=-4+12=8,\;y(-1)=4\) → \((8,\,4)\)

Example — Problem 30c: \(x(t)=-1+3\cos t,\;\;y(t)=1+2\sin t,\;\;t\in[0,2\pi)\)

\(\dfrac{dx}{dt} = -3\sin t\), \(\quad\dfrac{dy}{dt} = 2\cos t\)


Horizontal (\(dy/dt=0\)): \(\cos t=0 \Rightarrow t=\tfrac{\pi}{2},\,\tfrac{3\pi}{2}\)

\(t=\tfrac{\pi}{2}\): point \((-1,\;3)\)  |  \(t=\tfrac{3\pi}{2}\): point \((-1,\;-1)\)

Horizontal tangent points: \((-1,\,3)\) and \((-1,\,-1)\)


Vertical (\(dx/dt=0\)): \(\sin t=0 \Rightarrow t=0,\,\pi\)

\(t=0\): point \((2,\;1)\)  |  \(t=\pi\): point \((-4,\;1)\)

Vertical tangent points: \((2,\,1)\) and \((-4,\,1)\)

Extra Example: \(x(t)=2t^3-6t,\;\;y(t)=t^2-4t\) at \(t=3\)

\(\dfrac{dx}{dt}=6t^2-6\), \(\quad\dfrac{dy}{dt}=2t-4\)

At \(t=3\): \(dx/dt=48\), \(\;dy/dt=2\) → \(m = \dfrac{2}{48}=\dfrac{1}{24}\)

Point: \(x(3)=36,\;y(3)=-3\)

Tangent: \(y+3=\dfrac{1}{24}(x-36)\)
