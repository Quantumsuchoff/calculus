§1

Polar ↔ Rectangular Coordinate Conversions

Core Conversion Identities
x = r·cos(θ)    y = r·sin(θ)
r² = x² + y²    tan(θ) = y/x
r = √(x² + y²)    θ = arctan(y/x) [adjust quadrant!]

Algorithm A — Rectangular (x, y) → Polar (r, θ)

Step-by-Step Procedure
  1. Compute r = √(x² + y²).
  2. Compute reference angle α = arctan(|y/x|).
  3. Determine the correct quadrant of (x, y) and assign θ:
    • Q1: θ = α
    • Q2: θ = π − α
    • Q3: θ = π + α
    • Q4: θ = 2π − α (or −α)
  4. Write answer as (r, θ) or [r, θ].
  5. Check: Verify x = r·cos(θ) and y = r·sin(θ) reproduce originals.
⚠ Quadrant Rule: arctan(y/x) always gives an angle in (−π/2, π/2). Always inspect the signs of x and y to place θ in the correct quadrant.

Algorithm B — Polar (r, θ) → Rectangular (x, y)

Step-by-Step Procedure
  1. Compute x = r·cos(θ).
  2. Compute y = r·sin(θ).
  3. Simplify. Done.
Example — (x, y) = (−3, 3) → Polar

r = √(9 + 9) = 3√2

α = arctan(3/3) = arctan(1) = π/4

Point is in Q2 (x < 0, y > 0) → θ = π − π/4 = 3π/4

Answer: (3√2, 3π/4)

Polar ↔ Algebraic Equation Conversions

Algorithm C — Rectangular Equation → Polar

Step-by-Step Procedure
  1. Replace every x with r·cos(θ) and every y with r·sin(θ).
  2. Replace every x² + y² directly with r² (shortcut — do this first).
  3. Simplify and factor out r where possible.
  4. If r appears in the denominator, multiply both sides by r.
  5. Solve for r (or leave in implicit polar form).
Tip: Always scan for x² + y² first — replace it with r² before substituting x and y individually. This cuts algebra significantly.
Example — x² + (y − 4)² = 16 → Polar (Problem 23a)

Expand: x² + y² − 8y + 16 = 16

Simplify: x² + y² − 8y = 0

Replace x² + y² → r²: r² − 8y = 0

Replace y → r·sin(θ): r² − 8r·sin(θ) = 0

Factor: r(r − 8sin(θ)) = 0

Answer: r = 8sin(θ)

Example — (x − 7)² + y² = 49 → Polar (Problem 23b)

Expand: x² − 14x + 49 + y² = 49

Simplify: x² + y² − 14x = 0

Replace: r² − 14r·cos(θ) = 0

Factor: r(r − 14cos(θ)) = 0

Answer: r = 14cos(θ)

Example — x² + y² = 16x → Polar (Problem 23c)

Replace x² + y² → r² and x → r·cos(θ): r² = 16r·cos(θ)

Divide by r: r = 16cos(θ)

Example — x² + y² − 24y = 0 → Polar (Problem 23d)

Replace x² + y² → r² and y → r·sin(θ): r² − 24r·sin(θ) = 0

Factor: r(r − 24sin(θ)) = 0

Answer: r = 24sin(θ)

Algorithm D — Polar Equation → Rectangular

Step-by-Step Procedure
  1. Multiply both sides by r if needed to create r², r·cos(θ), or r·sin(θ) terms.
  2. Replace: r² → x²+y², r·cos(θ) → x, r·sin(θ) → y.
  3. Simplify and, if it's a circle, complete the square to identify center/radius.
Quick Reference — Common Polar Forms
r = a·cos(θ) → circle center (a/2, 0), radius |a/2|
r = a·sin(θ) → circle center (0, a/2), radius |a/2|
r = a → circle center (0,0), radius a
θ = c → line through origin at angle c

Identifying & Graphing Polar Curves

Classify the curve first, then graph it using a table of key (r, θ) values.

Polar Curve Identification Table

FormCurve TypeKey FeaturesProblem Examples
r = a Circle Centered at origin, radius a Prob 27: r = 3
r = a·cos(θ) or a·sin(θ) Circle Passes through origin; right/left or up/down 22A, 22F
r = a ± b·cos(θ), a/b ≥ 1 Limaçon (no inner loop) If a = b: cardioid. If a > b: dimpled or convex. 22B, 26a: r=2+4cos(θ)
r = a ± b·cos(θ), a/b < 1 Limaçon with inner loop Loops where r < 0 26a, 26b
r = a·cos(nθ) or a·sin(nθ) Rose / Flower n petals if n odd; 2n petals if n even 22D, 24: r=2cos(3θ), 25: r=2sin(2θ)
r = a·sin(θ) ± b (shifted) Cardioid / Limaçon Opens up or down 22B, 22C

Algorithm E — Graphing a Polar Curve

Step-by-Step Procedure
  1. Identify the type using the table above.
  2. Find zeros: Set r = 0 and solve for θ. These are angles where the curve passes through the origin.
  3. Find maximum |r|: Set dr/dθ = 0 or inspect the trig function for max/min.
  4. Build a table of (θ, r) at key angles: 0, π/6, π/4, π/3, π/2, 2π/3, 3π/4, π, and then by symmetry.
  5. Check symmetry:
    • Replace θ → −θ: symmetric about polar axis (x-axis)
    • Replace θ → π−θ: symmetric about θ = π/2 (y-axis)
    • Replace r → −r: symmetric about pole (origin)
  6. Plot the (r, θ) pairs in polar form. Connect smoothly.
  7. Note inner loops: they occur when r < 0 (the curve traces behind the pole).

Rose Curve Details — Problems 24 & 25

Rose Curve Rules
r = a·cos(nθ): n odd → n petals | n even → 2n petals
r = a·sin(nθ): same petal count rules; rotated by π/(2n)
Each petal's tip: r = a (maximum r)
Petal tips occur at: nθ = 0, π, 2π, ... (for cos) or nθ = π/2, 3π/2, ... (for sin)

Problem 25: r = 2sin(2θ) → 4-petal rose (n=2, even), petals at θ = π/4, 3π/4, 5π/4, 7π/4.

§4

Areas of Polar Regions

Master Area Formula — Polar Region
A = (1/2) ∫[α to β] r² dθ

Area between two curves (outer r₂, inner r₁):
A = (1/2) ∫[α to β] [r₂² − r₁²] dθ

Algorithm F — Area Enclosed by a Single Polar Curve

Step-by-Step Procedure (e.g. full rose petal, full circle)
  1. Identify the curve r = f(θ).
  2. Find the limits α and β by setting r = 0 and solving — these are the angles where the curve starts and ends one complete region.
  3. Use symmetry if possible (integrate over half then double).
  4. Set up: A = (1/2)∫[α to β] r² dθ.
  5. Expand r² (use power-reduction identities for trig²).
  6. Integrate term by term and evaluate.
Essential Trig Power-Reduction (needed for r²)
cos²(θ) = (1 + cos(2θ)) / 2
sin²(θ) = (1 − cos(2θ)) / 2
cos²(nθ) = (1 + cos(2nθ)) / 2

Algorithm G — Area of ONE Petal (Rose Curve)

Step-by-Step Procedure
  1. Find bounds: Set r = 0 → solve for θ → these give the angular bounds of one petal.
  2. For r = a·cos(nθ): one petal spans θ ∈ [−π/(2n), π/(2n)]. By symmetry, integrate 0 to π/(2n) and double.
  3. Write A_one petal = (1/2) ∫ r² dθ over those bounds.
  4. Apply power-reduction to cos²(nθ), integrate, evaluate.
  5. For total area of all petals: multiply A_one petal by the number of petals.
Example — Area enclosed by r = 2cos(3θ) [Problem 24]

One petal: cos(3θ) = 0 → 3θ = π/2 → θ = π/6. So petal: [−π/6, π/6].

A_one = (1/2)∫[−π/6 to π/6] (2cos3θ)² dθ = (1/2)∫4cos²(3θ) dθ = 2∫cos²(3θ) dθ

= 2∫[(1+cos(6θ))/2] dθ = ∫[1 + cos(6θ)] dθ = [θ + sin(6θ)/6] from −π/6 to π/6

= (π/6 + 0) − (−π/6 + 0) = π/3

3 petals total → Total Area = 3 × π/3 = π

Algorithm H — Inner Loop Area (Limaçon)

Step-by-Step Procedure
  1. Identify the limaçon r = a ± b·cos(θ) or r = a ± b·sin(θ) with a/b < 1.
  2. Find where r = 0: solve a ± b·cos(θ) = 0 → cos(θ) = ∓a/b. Let these be θ₁ and θ₂ (the inner loop bounds).
  3. The inner loop is traced for θ ∈ [θ₁, θ₂] (where r ≤ 0).
  4. Area of inner loop = (1/2)∫[θ₁ to θ₂] r² dθ.
  5. Note: r² is always positive, so the inner loop area formula is the same regardless of r's sign.

Algorithm I — Outer Loop / Between Loops

Step-by-Step Procedure
  1. Total area enclosed by the full curve = (1/2)∫[0 to 2π] r² dθ.
  2. Outer loop area = Total area − Inner loop area.
  3. Area between loops = Outer loop area − Inner loop area.
  4. Alternatively, set up outer loop bounds (where r ≥ 0) directly and integrate.

Algorithm J — Area Between Two Polar Curves (Problem 27)

Step-by-Step Procedure
  1. Find intersections: Set r₁ = r₂ and solve for θ. Also check if both curves pass through the origin.
  2. Determine which curve is the outer (larger r) on each interval.
  3. Set up: A = (1/2) ∫[α to β] [r_outer² − r_inner²] dθ
  4. For "inside r=3 and outside r=3−3cosθ": outer = r=3, inner = r=3−3cosθ, over the interval where 3 > 3−3cosθ, i.e., cosθ > 0.
  5. For "inside r=3−3cosθ and outside r=3": swap inner/outer.
Example — Problem 27: r = 3 and r = 3 − 3cos(θ)

Intersection: 3 = 3 − 3cos(θ) → cos(θ) = 0 → θ = π/2 and 3π/2.

(a) Area between both curves = (1/2)∫[−π/2 to π/2][3² − (3−3cosθ)²] dθ × 2 (by symmetry)

(b) Inside r=3, outside r=3−3cosθ: outer is r=3 on [−π/2, π/2]

A = (1/2)∫[−π/2 to π/2][9 − (3−3cosθ)²] dθ

(c) Inside r=3−3cosθ, outside r=3: outer is r=3−3cosθ on [π/2, 3π/2]

A = (1/2)∫[π/2 to 3π/2][(3−3cosθ)² − 9] dθ

Parametric Curves ↔ Algebraic Equations

Algorithm K — Eliminate the Parameter (→ Algebraic Equation)

Step-by-Step Procedure
  1. Look at x(t) and y(t) and choose a strategy:
  2. Strategy 1 — Direct substitution: Solve one equation for t, substitute into the other.
  3. Strategy 2 — Trig identity: If x = a·cos(t), y = b·sin(t), then (x/a)² + (y/b)² = cos²(t) + sin²(t) = 1.
  4. Strategy 3 — Exponent: If x = eᵗ and y = e^(kt), then y = xᵏ.
  5. Strategy 4 — Power: If x = t^m and y = t^n, then y = x^(n/m).
  6. State any restrictions on x and/or y imposed by the original parameter range.
Example — x(t) = sec²(t), y(t) = 6tan(t) [Problem 29a]

From y: tan(t) = y/6, so tan²(t) = y²/36.

Identity: sec²(t) − tan²(t) = 1 → x − y²/36 = 1

Answer: x = 1 + y²/36 (a parabola, x ≥ 1)

Example — x(t) = −1 + 4eᵗ, y(t) = 2 + e²ᵗ [Problem 29b]

From x: eᵗ = (x+1)/4, so e²ᵗ = ((x+1)/4)² = (x+1)²/16.

Substitute into y: y = 2 + (x+1)²/16

Answer: y = 2 + (x+1)²/16 (a parabola, x > −1)

Example — x(t) = −1 + 3cos(t), y(t) = 1 + 2sin(t) [Problem 30a]

cos(t) = (x+1)/3, sin(t) = (y−1)/2.

Apply cos²(t) + sin²(t) = 1:

Answer: (x+1)²/9 + (y−1)²/4 = 1 (ellipse, center (−1,1), a=3, b=2)

§6

Parametrization of Geometric Objects

Algorithm L — Parametrize a Line Segment from P₁ to P₂

Step-by-Step Procedure
  1. Let P₁ = (x₁, y₁) be the starting point and P₂ = (x₂, y₂) be the ending point.
  2. Compute direction vector: Δx = x₂ − x₁, Δy = y₂ − y₁.
  3. Write the parametrization with t ∈ [0, 1]:
    • x(t) = x₁ + (x₂ − x₁)·t
    • y(t) = y₁ + (y₂ − y₁)·t
  4. At t=0 you get P₁, at t=1 you get P₂. ✓
  5. Alternatively use any interval, e.g. t ∈ [0, 1] is standard.
Example — Line segment from (1, 6) to (−3, 1) [Problem 28a]

x(t) = 1 + (−3 − 1)t = 1 − 4t

y(t) = 6 + (1 − 6)t = 6 − 5t

Answer: x(t) = 1 − 4t, y(t) = 6 − 5t, t ∈ [0, 1]

Example — From (−2, 3) to (5, −1)

x(t) = −2 + 7t, y(t) = 3 − 4t, t ∈ [0, 1]

Algorithm M — Parametrize a Circle or Ellipse

Step-by-Step Procedure
  1. Circle x² + y² = r²: → x(t) = r·cos(t), y(t) = r·sin(t), t ∈ [0, 2π]
  2. Circle (x−h)² + (y−k)² = r²: → x(t) = h + r·cos(t), y(t) = k + r·sin(t)
  3. Ellipse x²/a² + y²/b² = 1: → x(t) = a·cos(t), y(t) = b·sin(t), t ∈ [0, 2π]
  4. Ellipse (x−h)²/a² + (y−k)²/b² = 1: → x(t) = h + a·cos(t), y(t) = k + b·sin(t)
  5. Read off a² and b² by rewriting the equation in standard form first.
Example — x² + y² = 4 [Problem 28b]

r = 2. x(t) = 2cos(t), y(t) = 2sin(t), t ∈ [0, 2π]

Example — 25x² + 9y² = 225 [Problem 28c]

Divide by 225: x²/9 + y²/25 = 1 → a² = 9, b² = 25, so a = 3, b = 5.

x(t) = 3cos(t), y(t) = 5sin(t), t ∈ [0, 2π]

Example — Ellipse 9x² + 16y² = 144

Divide by 144: x²/16 + y²/9 = 1 → a = 4, b = 3.

x(t) = 4cos(t), y(t) = 3sin(t), t ∈ [0, 2π]

§7

Tangent Lines & Slopes of Parametric Curves

Parametric Slope Formula
dy/dx = (dy/dt) / (dx/dt)    [provided dx/dt ≠ 0]

Horizontal tangent: dy/dt = 0 AND dx/dt ≠ 0
Vertical tangent: dx/dt = 0 AND dy/dt ≠ 0

Algorithm N — Tangent Line at a Specific t-Value

Step-by-Step Procedure
  1. Compute dy/dt and dx/dt (differentiate x(t) and y(t) with respect to t).
  2. Compute slope: m = (dy/dt) / (dx/dt).
  3. Evaluate m at the given t-value to get a number.
  4. Find the point: plug the t-value into x(t) and y(t) to get (x₀, y₀).
  5. Write the tangent line using point-slope form: y − y₀ = m(x − x₀).
Example — x(t) = 2t³ − 6t, y(t) = t² − 4t; tangent at t = 3

dx/dt = 6t² − 6  |  dy/dt = 2t − 4

At t=3: dx/dt = 6(9)−6 = 48  |  dy/dt = 6−4 = 2

Slope m = 2/48 = 1/24

Point: x(3) = 2(27)−18 = 36  |  y(3) = 9−12 = −3 → (36, −3)

Tangent line: y + 3 = (1/24)(x − 36)

Example — x(t) = −1 + 3cos(t), y(t) = 1 + 2sin(t); tangent at t = π/4 [Problem 30b]

dx/dt = −3sin(t)  |  dy/dt = 2cos(t)

At t=π/4: dx/dt = −3(√2/2) = −3√2/2  |  dy/dt = 2(√2/2) = √2

m = √2 / (−3√2/2) = √2 · (−2/(3√2)) = −2/3

Point: x = −1 + 3(√2/2) = −1 + 3√2/2  |  y = 1 + 2(√2/2) = 1 + √2

Tangent line: y − (1+√2) = −(2/3)(x − (−1 + 3√2/2))

Algorithm O — Find ALL Horizontal & Vertical Tangent Points

Step-by-Step Procedure
  1. Compute dx/dt and dy/dt.
  2. Horizontal tangents: Solve dy/dt = 0 for all t in the domain. For each solution t₀, verify dx/dt ≠ 0 at t₀. Then compute (x(t₀), y(t₀)).
  3. Vertical tangents: Solve dx/dt = 0 for all t in the domain. For each solution t₀, verify dy/dt ≠ 0 at t₀. Then compute (x(t₀), y(t₀)).
  4. Report the actual (x, y) coordinate pairs, not just the t-values.
⚠ Caution: If both dx/dt = 0 AND dy/dt = 0 at the same t, the tangent is indeterminate (need L'Hôpital or higher-order analysis). These are called cusps or singular points.
Example — x(t) = 4t³ − 12t, y(t) = t² + 3 [Problem 31]

dx/dt = 12t² − 12 = 12(t² − 1) = 12(t−1)(t+1)

dy/dt = 2t


Tangent line at t = 2:

m = (2·2)/(12·4 − 12) = 4/36 = 1/9

Point: x(2) = 32 − 24 = 8, y(2) = 4+3 = 7 → (8, 7)

Line: y − 7 = (1/9)(x − 8)


Horizontal tangents (dy/dt = 0): 2t = 0 → t = 0.

Check: dx/dt at t=0 = 12(0−1) = −12 ≠ 0. ✓

Point: x(0) = 0, y(0) = 3 → (0, 3)


Vertical tangents (dx/dt = 0): t = 1 or t = −1.

At t=1: dy/dt = 2 ≠ 0 ✓ → x(1) = 4−12 = −8, y(1) = 4 → (−8, 4)

At t=−1: dy/dt = −2 ≠ 0 ✓ → x(−1) = −4+12 = 8, y(−1) = 4 → (8, 4)

Example — x(t) = −1 + 3cos(t), y(t) = 1 + 2sin(t) [Problem 30c]

dx/dt = −3sin(t)  |  dy/dt = 2cos(t)


Horizontal (dy/dt = 0): cos(t) = 0 → t = π/2, 3π/2

Check dx/dt: at t=π/2, −3sin(π/2) = −3 ≠ 0 ✓

t=π/2: (−1+0, 1+2) = (−1, 3)  |  t=3π/2: (−1+0, 1−2) = (−1, −1)

Horizontal tangent points: (−1, 3) and (−1, −1)


Vertical (dx/dt = 0): sin(t) = 0 → t = 0, π

Check dy/dt: at t=0, 2cos(0) = 2 ≠ 0 ✓

t=0: (−1+3, 1+0) = (2, 1)  |  t=π: (−1−3, 1+0) = (−4, 1)

Vertical tangent points: (2, 1) and (−4, 1)
